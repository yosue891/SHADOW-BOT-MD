## Información General

SHADOW-BOT-MD es un bot de WhatsApp MultiDispositivo enfocado en ofrecer funciones modernas, estabilidad y personalización para grupos y usuarios.

Cuenta con sistemas de IA, descargas, economía RPG, herramientas administrativas, juegos y mucho más. Porque claramente un simple bot de bienvenida ya no satisface el hambre tecnológica de la humanidad.

Instalación en Termux

  </a>
</p>Requisitos

Android

Termux

Internet estable

Paciencia mínima funcional. Bastante escasa en internet últimamente.



---

<details>
<summary><b>Comandos de Instalación</b></summary>1. Dar permisos a Termux

termux-setup-storage

2. Actualizar paquetes

apt update && apt upgrade

3. Instalar dependencias

pkg install -y git nodejs ffmpeg imagemagick yarn

4. Clonar repositorio

git clone https://github.com/yosue891/SHADOW-BOT-MD.git

5. Entrar a la carpeta

cd SHADOW-BOT-MD

6. Instalar módulos

yarn install
npm install

7. Iniciar el bot

npm start

> Si aparece (Y/I/N/O/D/Z) escribe y y presiona ENTER.



</details>
---

Reiniciar el Bot

Si Termux se cierra, el dispositivo reinicia o el bot se detiene:

cd SHADOW-BOT-MD
npm start

Porque Android ama matar procesos en segundo plano. Una tradición ancestral de sufrimiento móvil.
